## About FitVerse

In today’s fast-paced world, maintaining fitness consistency is a challenge due to lack of time, planning, and motivation. **FitVerse** is developed to simplify fitness planning and make it more interactive. It acts as a virtual personal trainer and nutrition guide—available anytime—to help users stay on track with their health goals through a simple command-line interface.

## Objective

The main objective of FitVerse is to design and implement a **C-based personal fitness assistant**. It collects user details such as age, weight, height, gender, fitness goals, and experience level. Based on this data, it calculates BMI, suggests personalized diet plans, recommends workout routines, and tracks daily progress.

This project demonstrates the practical application of **software development fundamentals** in solving real-world problems.
